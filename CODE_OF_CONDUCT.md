# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating — opening issues, submitting PRs, joining Discussions,
or any other interaction within the Mnela community — you agree to
uphold its terms.

## Reporting

If you witness or experience a violation, please report it via the
private [security advisory form](https://github.com/SmartDogg/mnela/security/advisories/new)
or by emailing the maintainer through the contact on the
[GitHub profile](https://github.com/SmartDogg). All reports are
handled confidentially.
